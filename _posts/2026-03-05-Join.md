## 1. JOIN의 정의 및 핵심 차이
**Join**이란 두 개 이상의 테이블을 서로 특정 컬럼(주로 PK-FK)을 기준으로 연결하여 하나의 통합된 결과 집합을 만드는 연산.

* **Inner Join (내부 조인)**: 두 테이블 모두에 공통된 데이터가 존재하는 행만 결합하여 보여주는 방식.
* **Left Outer Join (왼쪽 외부 조인)**: 왼쪽 테이블의 모든 데이터를 유지하면서, 오른쪽 테이블에서 매칭되는 데이터가 있는 경우에만 결합함. 매칭되는 데이터가 없으면 `NULL`로 표시됨.

조인 연산을 설명하기 위한 기본 테이블 구조.

### **video 테이블**
| id | title | y_id |
| :--- | :--- | :--- |
| 1 | 데이터베이스 완전정복 | 2 |
| 2 | 볼리비아 광산 탐방기 | 4 |

| id | name | 채널 설명 |
| :--- | :--- | :--- |
| 1 | 쯔양 | 먹방 |
| 2 | 빠니보틀 | 여행 |
| 3 | 침착맨 | 예능 |

## 3. Inner Join (내부 조인) 결과
두 테이블에 모두 데이터가 존재하는(y_id와 id가 일치하는) 행만 검색.

```sql
SELECT * FROM video INNER JOIN youtuber ON video.y_id = youtuber.id;
