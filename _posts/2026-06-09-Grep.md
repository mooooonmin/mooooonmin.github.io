## 1. `grep`이란?

`grep`은 입력된 내용 중에서 특정 키워드나 패턴이 들어간 줄만 찾아 출력하는 명령어다.

GNU grep 매뉴얼은 `grep`을 패턴과 일치하는 줄을 출력하는 도구라고 설명한다. [1]
또한 입력 파일이 지정되지 않으면 표준 입력에서 검색한다고 설명한다. [2]

그래서 `grep`은 단독으로도 사용할 수 있지만, 다른 명령어와 조합해서 사용하는 경우가 많다.
특히 `|` 파이프라인을 사용하면 앞쪽 명령어의 출력 결과 중에서 원하는 줄만 골라낼 수 있다.

기본 형태는 아래와 같다.

```bash
[명령어] | grep [찾을 키워드]
```

---

## 2. 파이프라인과 함께 사용하는 이유

`|` 파이프라인은 앞쪽 명령어의 표준 출력(stdout)을 뒤쪽 명령어의 표준 입력(stdin)으로 넘긴다.

예를 들어 아래 명령어는 `/` 디렉터리의 파일 목록을 자세히 출력한다.

```bash
ls -al
```

여기서 `media`가 들어간 줄만 보고 싶다면 `grep`을 함께 사용한다.

```bash
ls -al | grep media
```

이 명령어의 흐름은 아래와 같다.

| 단계 | 동작 |
|---|---|
| `ls -al` | `/` 디렉터리의 파일 목록을 출력한다. |
| `|` | 앞 명령어의 출력 결과를 뒤 명령어로 전달한다. |
| `grep media` | 전달받은 줄 중 `media`가 들어간 줄만 출력한다. |

즉, `grep`은 앞 명령어의 출력 결과 전체를 다시 필터링하는 역할을 한다.

---

## 3. 예제 1: `ls -al` 결과에서 찾기

루트 디렉터리로 이동한 뒤 파일 목록을 확인한다.

```bash
cd /
ls -al
```

출력 결과 중 `media`가 들어간 줄만 보고 싶다면 아래처럼 입력한다.

```bash
ls -al | grep media
```

출력 결과 중 `bin`이 들어간 줄만 보고 싶다면 아래처럼 입력한다.

```bash
ls -al | grep bin
```

`grep media`는 전체 출력 결과 중 `media`라는 문자열이 포함된 줄만 보여준다.
`grep bin`은 전체 출력 결과 중 `bin`이라는 문자열이 포함된 줄만 보여준다.

---

## 4. 예제 2: 파일 내용에서 찾기

이번에는 파일 내용 중 특정 키워드가 들어간 줄만 찾아보자.

```bash
cd /var/log
cat README
```

`cat README`는 `README` 파일 내용을 터미널 화면에 출력한다.
이 출력 결과 중 `Alternatively`가 들어간 줄만 보고 싶다면 아래처럼 입력한다.

```bash
cat README | grep Alternatively
```

`you`가 들어간 줄만 보고 싶다면 아래처럼 입력한다.

```bash
cat README | grep you
```

이 방식은 파일 내용이 길 때 유용하다.
전체 내용을 모두 읽지 않고, 원하는 키워드가 포함된 줄만 빠르게 확인할 수 있다.

## 5. 예제 3: 설치된 패키지 목록에서 찾기

이전 `apt` 명령어를 설명할 때도 `grep`을 사용했다.

현재 컴퓨터에 설치된 모든 패키지를 확인하려면 아래 명령어를 사용할 수 있다.

```bash
sudo apt list --installed
```

하지만 출력 결과가 많기 때문에 원하는 패키지를 찾기 어렵다.
이럴 때 `grep`을 함께 사용한다.

설치된 패키지 중 `python`이 들어간 줄만 보고 싶다면 아래처럼 입력한다.

```bash
sudo apt list --installed | grep python
```

설치된 패키지 중 `zstd`가 들어간 줄만 보고 싶다면 아래처럼 입력한다.

```bash
sudo apt list --installed | grep zstd
```

이 명령어의 의미는 아래와 같다.

| 명령어 | 의미 |
|---|---|
| `sudo apt list --installed` | 설치된 패키지 목록을 출력한다. |
| `grep python` | 출력 결과 중 `python`이 들어간 줄만 출력한다. |
| `grep zstd` | 출력 결과 중 `zstd`가 들어간 줄만 출력한다. |

## 6. `grep`을 사용할 때 기억할 점

입문 단계에서는 `grep`을 아래처럼 이해하면 충분하다.

```bash
[명령어] | grep [찾을 키워드]
```

이 형태는 앞쪽 명령어의 출력 결과 중에서 특정 키워드가 들어간 줄만 보여준다.

예를 들어 아래 명령어들은 모두 같은 패턴이다.

```bash
ls -al | grep media
cat README | grep you
sudo apt list --installed | grep python
```

`grep`은 다른 명령어와 함께 조합해서 사용하는 경우가 많다.
그래서 Linux 명령어를 사용할 때 반드시 익숙해져야 하는 명령어 중 하나다.

## 핵심 정리

`grep`은 입력된 내용 중 특정 키워드나 패턴이 들어간 줄만 찾아 출력하는 명령어다.

다른 명령어와 함께 사용할 때는 보통 아래 형태로 작성한다.

```bash
[명령어] | grep [찾을 키워드]
```

`|` 파이프라인은 앞 명령어의 출력 결과를 뒤 명령어로 넘긴다.
`grep`은 넘겨받은 출력 결과 중에서 키워드가 포함된 줄만 출력한다.

처음에는 아래 예시를 통째로 기억해두면 된다.

```bash
ls -al | grep media
cat README | grep you
sudo apt list --installed | grep python
```
